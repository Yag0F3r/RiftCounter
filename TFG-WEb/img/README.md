# Carpeta de Imágenes

Esta carpeta contiene las imágenes que se mostrarán en el marcador deportivo.

## Imágenes por defecto:
- `default1.png` - Imagen para zona superior izquierda
- `default2.png` - Imagen para zona superior derecha  
- `default3.png` - Imagen para zona inferior izquierda
- `default4.png` - Imagen para zona inferior derecha

## Cómo cambiar las imágenes:

### Método 1: Reemplazar archivos
1. Coloca tus imágenes en esta carpeta
2. Renómbralas con los nombres exactos: `default1.png`, `default2.png`, etc.
3. Recarga la página web

### Método 2: Durante el juego
1. Haz click en cualquier imagen
2. Selecciona una nueva imagen desde tu dispositivo
3. La imagen se cambiará temporalmente (hasta recargar la página)

## Formatos soportados:
- PNG (recomendado)
- JPG/JPEG
- GIF
- WebP

## Tamaño recomendado:
- **Ancho**: 150-200 píxeles
- **Alto**: 200-300 píxeles
- **Proporción**: Vertical (más alto que ancho)

## Consejos:
- Usa imágenes con buena resolución pero no muy pesadas
- Las imágenes se redimensionarán automáticamente
- Si no encuentra la imagen, mostrará un placeholder de ejemplo
